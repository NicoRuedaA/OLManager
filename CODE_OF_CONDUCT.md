# Code of Conduct

OLManager uses the Contributor Covenant Code of Conduct as the baseline for community behavior.

## Our pledge

We pledge to make participation in this project welcoming, respectful, and harassment-free for contributors and users regardless of background, identity, experience level, or viewpoint.

## Expected behavior

- Be respectful and constructive.
- Assume good intent, but accept correction when impact differs from intent.
- Keep technical disagreement focused on evidence and project goals.
- Credit sources, contributors, and upstream projects accurately.

## Unacceptable behavior

- Harassment, insults, threats, or discriminatory language.
- Publishing private information without permission.
- Sustained disruption of issues, PRs, or Discussions.
- Ignoring maintainer moderation decisions.

## Enforcement

Maintainers may edit, hide, lock, or remove comments; close issues or PRs; temporarily restrict participation; or permanently ban participants when needed to protect the project community.

## Reporting

Report conduct concerns privately to the maintainer/security contact listed in [`MAINTAINERS.md`](MAINTAINERS.md). If that file still contains a placeholder, contact the repository owner through GitHub until a public address is confirmed.
