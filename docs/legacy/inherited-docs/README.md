# Inherited OpenFootManager Documentation

This directory preserves selected documentation inherited from the original OpenFootManager repository for historical reference and future archaeology.

These files are **not** current OLManager contributor or player documentation. They may contain old product names, football-era mechanics, stale test counts, outdated file paths, and unaudited data/provenance assumptions. Treat them as reference material only unless a future PR re-verifies and promotes specific content back into the current documentation set.

Current public documentation starts at [`../../README.md`](../../README.md). The disposition rationale for each moved or deleted inherited file is recorded in [`../../INHERITED_DOCS_AUDIT.md`](../../INHERITED_DOCS_AUDIT.md).
